# BetterRest

## アプリ仕様
BetterRestは起床予定時刻、睡眠時間、コーヒーの摂取数のデータから推奨就寝時刻を教えるアプリです。
起床予定時刻、睡眠時間、コーヒーの摂取数の値を入力すると、Core MLに渡され、理想的な就寝時刻を教えてくれます。
